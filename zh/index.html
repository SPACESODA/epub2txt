使用 epub2txt
將 EPUB 高效地轉為純文字

這是一個強大的開源 Python 腳本，可將整個電子書庫批量轉換為清晰、易讀且適合 LLM 使用的純文字 (TXT)。

前往腳本 線上轉換
⚡️

極速轉換

利用高效的流式提取技術，瞬間處理數百本書，極低內存佔用。

📂

批量處理

一個指令即可轉換單個檔案、檔案列表或整個資料夾。

✨

自動排版

自動保留結構並增加段落間距，提供清晰舒適的閱讀體驗。

簡單易用

隨附的輔助腳本已準備好，依照 README 的指示即可在幾個步驟內執行 Python 腳本。

轉換單個檔案
$ ./run.sh my_book.epub
轉換整個資料夾
$ ./run.sh /Books/SciFi/ /Books/History/
Scanning folder: /Books/SciFi/
Found 12 EPUBs.
...

線上轉換

採用與 Python 腳本相同的邏輯，直接在你的瀏覽器執行轉換。可轉換單一或多個檔案！
